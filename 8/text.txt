08. Цитаты в HTML — теги blockquote, cite. Атрибут cite. Источник цитаты в HTML

Тег blockquote.
Является блочным тегом, а значит растягивается на всю доступную ширину. 
Служит для большого блока с цитатой.
Внутри обычно содержит текст, который обычно обрернут в тег p.
Цитата без указания источника - плохая цитата с точки зрения
информативности и семантики. Есть два споособа, чтобы указать источник:
1. Обернуть источник в тег cite
2. В теге blockquote указать атрибут cite="<адрес>" 

Тег site - строчный тег, которым оборачивают текст с источником цитаты.
Размещают его вне тега blockquote, сразу после него. 

<blockquote>
        <p>
            Lorem ipsum, dolor sit amet consectetur adipisicing elit.
        </p>
    </blockquote>
    <p>Имя автора, <cite>Источник...</cite></p>

Источником для цитаты является ресурс, откуда была взята цитата,
то есть там, где была опубликована.

Тег site может быть использован не только для указания цитат, но а также для названий книг,
статей, картин, объектов искусства или даже ссылаться на специфичный раздел сайта.

Атрибут site.

<blockquote site="https://some-site.com">
        <p>
            Lorem ipsum, dolor sit amet consectetur adipisicing elit.
        </p>
    </blockquote>

Отличие в этих способах в том, что при использовании тега site текст источника
буден виден на странице, а с атрибутом site - нет, в этом случае мы просто увеличим семантику разметки.


Тег q.
Строчный тег, аналог блочного тега blockquote.
Используется для выделения коротких цитат прямо в тексте, например внутри параграфа.
При использовании добавятся двойные кавычки, которые можно стилизовать через CSS.
