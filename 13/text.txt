13. Формы в HTML. Теги textarea, select, option, optgroup

Тег textarea.
Выводит многострочное поле ввода. Парный тег.

<label for="comment">Ваш комментарий: </label>
<textarea id="comment"></textarea>

Если написать как-то текст между открывающим и закрывающим тегом, 
текст станет значением элемента textarea по умолчанию.
Такая запись аналогична использованию атрибуту value в теге input.

<label for="comment">Ваш комментарий: </label>
<textarea id="comment">Это значение по умолчанию</textarea>

Браузер по умолчанию разрешает менять ширину и высоту поля, потянув за нижний угол.
Такое поведение можно отлючить по любом осям через CSS.

Тег select.
Используется для выбора из выпадающего списка одного или нескольких 
заранее подготовленных опций.
Дочерними элементами тега select являются теги option, каждый из которых
является одним значением из вариантов ответа.
Для option обязательным является заполнение атрибута value, который необходим
серверу, для того, чтобы распознать, что выбрал пользователь после того,
как форма отправилась.

<label for="city">Выберите город:</label>
<select id="sity">
    <option value="moscow">Москва</option>
    <option value="krasnodar">Краснодар</option>
    <option value="perm">Пермь</option>
    <option value="saratov">Саратов</option>
</select>

Атрибут multiple.
Чтобы в списке можно было выбрать более одного ответа, нужно использовать
атрибут multiple для тега select. В таком случае выпадющий список превращается
в раскрытый список.

<label for="city">Выберите город:</label>
<select id="sity" multiple>
    <option value="moscow">Москва</option>
    <option value="krasnodar">Краснодар</option>
    <option value="perm">Пермь</option>
    <option value="saratov">Саратов</option>
</select>

Атрибут selected.
По умолчанию выбрано значение первой опции.
Для того, чтобы изменить значение по умолчанию, добавьте атрибут selected
в option с нужным значением.

<label for="city">Выберите город:</label>
<select id="sity">
    <option value="moscow">Москва</option>
    <option value="krasnodar">Краснодар</option>
    <option value="perm">Пермь</option>
    <option value="saratov" selected>Саратов</option>
</select>


Тег select без выбранной по умолчанию опции.
Допустим, нам нужно сделать так, чтобы в select по умолчанию не было выбрано
не одной опции.
Во-первых, нужно убедиться, что ни у одной опции нет атрибута selected.
Во-вторых, добавить новую опцию с названием. 
Но остается одна проблема - если выбор в select обязателен (атрибут required),
то опция "Не выбрано" будет валидным вариантом и пользователь сможет отрпавить форму,
ничего не выбирая.
Есть хитрость, как решить эту проблему. Добавляем атрибут value с пустым значением.
Теперь при попытке отправить форму, появится предупреждение о том,
что нужно выбрать одну опцию в селекте.
В целом мы можем убрать соблазн для пользователя выбрать значение "Не выбрано",
присвоив атрибут disabled этой опции. 
Однако теперь при disabled следующая опция становится выбранной.
А это не то, что нам хотелось бы.
Но проблема легко решаема, добавляет атрибут selected для опции с пустым значением.

В результате все работает так, как нам надо.
При загрузке страницы пользователь видит значение по умолчанию.
Не выбрав никакую опцию отрпавить форму нельзя.
Пользователь вынужден выбрать значение.

Стилизовать классический браузерный селект не выйдет.
Либо использовать готовые компоненты, либо делать свою реализацию с разметкой, 
с примененим JS. Это очень интересно и полезно для прокачки скиллов.
Убрать стрелку можно и через background-image можно поставить свою.

Тег optgroup.
Используется для визуальной группировки нескольких тегов option.
В атрибуте label указывается заголовок группы.

<label for="city">Выберите город:</label>
<select id="sity" required>
    <option value="" disabled selected>Не выбран</option>
    <optgroup label="Первая пачка">
        <option value="moscow">Москва</option>
        <option value="krasnodar">Краснодар</option>
    </optgroup>
    <optgroup label="Вторая пачка">
        <option value="perm">Пермь</option>
        <option value="saratov">Саратов</option>
    </optgroup>
</select>

В реальной практике этот тег встречается не часто, но знать его не будет лишним.
