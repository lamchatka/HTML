12. HTML тег input и его вариации. 
Атрибут type, значения tel, email, password, url, search и другие


Атрибут type со значением email.
Нужно для обозначения поля ввода адреса электронной почты.
Перед отправкой браузер проверить корретность введенного значения email,
а именно: наличие @ и . (домен).
На мобильных устройствах на клавиатуре будут кнопки собака (@) и точка (.)

<form>
    <p>
        <label for="email">Email:</label>
        <input id="email "type="email" name="user-name">
    </p>
    <p>
        <button type="submit">Отправить</button>
    </p>
</form>

Атрибут type со значением url.
Нужен для обозначения поля ввода url-адреса какой-го либо ресурса в интернете.
Перед отправкой браузер проверит корректность 
значения на соответствие формату url-адреса.
На мобильном устройстве на клавиатуре подставить вспомогательные кнопки: 
".", "/", ".com".

<form>
    <p>
        <label for="url">Url:</label>
        <input id="url "type="url" name="user-url">
    </p>
    <p>
        <button type="submit">Отправить</button>
    </p>
</form>

Атрибут type со значением tel.
Служит для обозначения поля ввода номера телефона.
Браузер не валидирует введенное значение потому, что форматов в мире очень много,
в отличие от type со значением email и type со значением url.
На моб. устр. подставит удобную клавиатуру с цифрами.

<form>
    <p>
        <label for="tel">tel:</label>
        <input id="tel" type="tel" name="user-tel">
    </p>
    <p>
        <button type="submit">Отправить</button>
    </p>
</form>

Атрибут type со значением search.
Служит для обозначения поля ввода поискового запроса.
Браузер выведет кнопку с крестиком для очистки введенного значения.
Крестик можно скрыть через CSS.
На моб. устр появится клавиатура с кнопкой seacrh.

<form>
    <p>
        <label for="search-query">search-query:</label>
        <input id="search-query "type="search" name="user-search-query">
    </p>
    <p>
        <button type="submit">Отправить</button>
    </p>
</form>


Атрибут type со значением number.
Служит для обозначения поля ввода числового значения.
Браузер выведет доп. элементы управления - стрелки вверх и вниз.
Стрелка вверх увеличит значение на 1, вниз - уменьшит.
Стрелки можно убрать через CSS.
На моб. устр появится клавиатура с цифрами и спец. символами.
 
<form>
    <p>
        <label for="age">Age:</label>
        <input id="age" type="number" name="user-age">
    </p>
    <p>
        <button type="submit">Отправить</button>
    </p>
</form>

Атрибут type со значением password.
Служит для поля ввода пароля.
Браузер будет скрывать вводимое в поле значение символом точки.

<form>
    <p>
        <label for="password">password:</label>
        <input id="password" type="password" name="user-password">
    </p>
    <p>
        <button type="submit">Отправить</button>
    </p>
</form>

Атрибут type для дат и времени.
Служит для обозначения поля ввода даты/времени.
Интерактивное окно не стилизуется средствами CSS.

Атрибут type со значением color.
Служит для выбора пользователем цвета с помощью встроенного
в браузер компонент цвета color picker.
Стилизовать через CSS нельзя.

Атрибут type со значением file.
Используют для прикрепления к форме файла. 
Чтобы прикрепить несколько файлов добавить атрибут multiple.


<form>
    <p>
        <label for="form-file">

            Выберите файл:
        </label>
    </p>
    <p>
        <input type="file" id="form-file" name="form-file">
    </p>
</form>

Атрибут type со значением checkbox.
Превращает тег input в другой элемент - квадрат, при клике на который
появляется галочка. Подобный элемент используется в связке с тегом label.
указывайте value, чтобы сервис считал значение чекбокса.
Атрибут checked ставит галочку по умолчанию.

<p>
    <input type="checkbox" id="agremeent" name="agremeent value="agremeent">
    <label for="agremeent">
        Я точно согласен с какими-то условиями
    </label>   
</p>


Атрибут type со значением radio.
Служит для выбора одного значения из нескольких предложенных.
Для корректной работы связать с label и указать атрибут name, 
значение которого должно быть уникально для всей группы,
среди которых пользователь может выбрать одно значение. ((???))
Короче группирует по name.
value также для сервера.
атрибут checked - выбор по умолчанию.


Атрибут type со значением range.
Выводит ползунок для выбора чисел из заданного диапазона.
Для задания границ диапазона используются атрибуты min, max.
Стилизовать этот элемент нельзя.
