05.  Форматирование текста в HTML — теги b, strong,
i и em, s, u, sub и sup, small, mark, br, hr, code

Для того, чтобы менять участки текста применяются теги форматирования.
Все теги форматирования текста являются строчными элементами
и их можно использовать с друг другом в любых комбинациях.

Теги b и strong. 
Используются, чтобы сделать текст более насыщенным - жирным.
Различие - тег b выделяет участок более жирным, не придавая никакой важности,
а strong помимо визуального эффекта добавляет важности - 
считывается screen ридерами с другой интонацией.

Теги i и em.
Используются для выделения участка текста курсивом.
Курсивов в тексте могут обозначать смену тона голоса, другие языки или настроение.
Разница между этими тегами аналогична ситуации с тегами b и strong - 
i добавляет только визуальный эффект, а em придает важности для вспомогательных программ.

Тег s.
Испольуется для выделения текста, который уже не актуален, например старая цена.
Визуально выглядит как перечеркнутый текст.

Тег u.
Используется для выделения фрагмента текста, который стилистически отличается от остального текста.
Например, слова с нарочитыми орфографическими ошибками.

Тег sub.
Используется, чтобы сделать текст подстрочным. Например, в химических формулах.

Тег sup.
Используется, чтобы сделать текст надстрочным. Например, степень числа в формуле.

Тег small.
Задает выделенному тексту мелкий размер шрифта. Используется для комментариев, пояснений 
правовых формулировок. Размер выводимого текста по умолчанию на 20% ниже основого.

Тег mark.
Выделяет фрагмент текста ярким фоном. По умолчанию желтый цвет. 
Используется при поиске на странице, чтобы выделить результат поиска.

Тег br.
Перенос на новую строку. Используется там, где важен перенос: в слоганах, в стихах, в адресе.
Используется довольно редко!!!!

Тег hr.
Разделительная линия между частями контента. Разбивает контент на смысловые группы.

Тег code.
Делаем выделенный фрагмент моноширийным - каждый символ в тексте имеет одинаковую ширину.

По умолчанию браузер игнорирует пробелы и переносы строк в HTML, 
позволяя писать так, как нам удобно.
Если нужно отобразить текст так, как мы его написали? Сохраняя пробелы и переносы строк.
Использовать тег pre. Сохраняет пробелы и переносы строк в браузере.
